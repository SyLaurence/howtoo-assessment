# HowToo Assessment Part 3

## How to Install

Note: Make sure [node](https://nodejs.org/en/download) is installed on your machine!

1. Clone the repository in your project directory.

```
git clone https://github.com/SyLaurence/howtoo-assessment.git
```

2. Go to the project folder.

```
cd howtoo-assessment
```

3. Install dependencies.

```
npm i
```

4. Run Dev Server.

```
npm run dev
```

5. Start the local server and go to [localhost:8080](http://localhost:8080/)

## Tech Stack

- Node
  - For basic server setup.
- Webpack
  - Module bundler and transpiler.
- React
  - JS Library
- React Router
  - For Page navigation (SPA)
- Javascript
- HTML / CSS
